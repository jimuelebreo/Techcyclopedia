To make this "Techcyclopedia" website work, first you have make a folder named "techcyclopedia" in this specific folder path (reminder you need to have the application "xampp") "C:/xampp/htdocs" and put all the file inside it.
Next start the xampp application and start Apache and MySql and open the admin for MySql, make a new database named "component_management_db" and upload the file named "component_management_db.sql". After that you're all set, 
just open a web browser and search for "http://localhost/techcyclopedia/index.html".
